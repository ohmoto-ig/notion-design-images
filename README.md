# notion-design-images

社内「設計・部品知識」Notion（ダークモード運用）に貼る参考図の画像ファイル置き場です。
Notion から `raw.githubusercontent.com` 経由で安定表示させるために、自由ライセンスの図のみを再ホストしています。

- 画像はすべて Wikimedia Commons などの自由ライセンス（CC BY-SA / CC BY / CC0 / パブリックドメイン）由来です。
- 出典・元ファイルは `CREDITS.md` を参照してください。
- 社外秘情報・Notion本文・図番等は含みません（画像ファイルのみ）。
